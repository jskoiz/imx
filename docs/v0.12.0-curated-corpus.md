# IMX v0.12.0 Curated Corpus Plan

The curated corpus is generated or embedded in tests. IMX does not vendor
unclear-license external image files for v0.12.0.

## Fixture Families

- Generated release fixtures from `imx-generate-fixtures`: FARBFELD RGBA16, QOI
  RGB linear, PPM comments with high `maxval`, PGM16, and PNG RGBA16.
- In-test fixtures in `tests/curated_corpus.rs`: PNG grayscale-alpha,
  progressive grayscale JPEG, PBM comments, malformed inputs, and resource
  boundaries.
- Fuzz seed copies from `scripts/seed-fuzz-corpus.sh`: selected generated intake
  fixtures are copied into FARBFELD, QOI, PNG, and PNM decode corpora.

## Evidence

- `cargo test --test curated_corpus`
- `bash scripts/curated-corpus.sh`
- `target/curated-corpus/summary.json`
- generated fixture `manifest.json`

## Boundaries

The corpus only increases confidence for existing supported formats and command
forms. It does not add new format support, metadata preservation, transforms,
streaming, or full ImageMagick CLI behavior.
