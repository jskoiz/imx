# IMX v0.12.0 Real-World Intake Reliability Slice

v0.12.0 keeps the v0.11.0 command and format surface unchanged and adds a
bounded reliability pass for already-supported inputs. The goal is better proof
and clearer failures for real-world intake, not new format breadth.

## Contract

- Supported formats remain FARBFELD, JPEG, QOI, PBM, PGM, PNG, and PPM.
- Supported commands remain `imx --help`, `imx --version`,
  `imx identify [FORMAT:]<input>`, and two-path transcodes with exact uppercase
  `FARBFELD:`, `JPEG:`, `QOI:`, `PBM:`, `PGM:`, `PNG:`, and `PPM:` prefixes.
- Intake coverage includes generated or in-test cases for FARBFELD RGBA16,
  progressive grayscale JPEG, QOI RGB linear, PBM comments, PGM scaled/16-bit
  samples, PNG grayscale-alpha/RGBA16, and PPM high `maxval` with comments.
- Malformed input failures should include useful codec detail and, at the CLI,
  operation, format, and path context.
- Resource limits remain policy, not ImageMagick parity: decoded pixel buffers
  are capped at 512 MiB, JPEG decoded buffers at 128 MiB, and CLI input reads at
  513 MiB.

## Proof Requirements

- `cargo test --test curated_corpus` passes.
- `scripts/curated-corpus.sh` emits `target/curated-corpus/summary.json`.
- Malformed tests prove explicit PNM sample errors, PBM sample errors, QOI
  truncated-opcode EOF accounting, and uppercase FARBFELD header diagnostics.
- CLI tests prove operation/path context for identify, decode, and encode
  failures.
- Prefix tests prove mismatched input/output prefixes fail across the supported
  matrix and that prefixes do not select extensionless outputs or add aliases.
- Fuzz seeds include the v0.12 generated intake fixtures.
- Install, package, release-archive, conformance, and Homebrew formula smoke
  include v0.12 intake checks.

## Still Out Of Scope

- No new image formats.
- No stdin/stdout, batch mode, resize/crop/filter transforms, or full
  ImageMagick CLI parsing.
- No metadata preservation, EXIF writing, ICC/profile transforms, color
  management, delegates, MagickCore, or MagickWand.
- No Homebrew/core, crates.io, Windows, or hosted macOS/iOS Actions.
