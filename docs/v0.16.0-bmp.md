# IMX v0.16.0 Bounded BMP Support

`v0.16.0` adds one bounded format slice: uncompressed Windows BMP.

## Supported Surface

- `imx identify input.bmp` and `imx identify BMP:input.bmp`.
- `imx input.bmp output.{bmp,ff,jpg,qoi,pbm,pgm,png,ppm}`.
- `imx input.{ff,jpg,qoi,pbm,pgm,png,ppm} output.bmp`.
- `imx resize <width>x<height> BMP:input.bmp BMP:output.bmp`.
- `imx resize-fit <width>x<height> BMP:input.bmp BMP:output.bmp`.
- `imx batch-convert --to BMP --output-dir <dir> [FORMAT:]<input>...`.
- Deterministic same-format BMP rewrites when input and output paths differ.

## BMP Contract

- File magic must be `BM`.
- The DIB header must be Windows `BITMAPINFOHEADER` (`biSize == 40`).
- Compression must be `BI_RGB` (`0`), planes must be `1`, and color tables are
  rejected.
- Decode supports 24-bit BGR rows as RGB8 and 32-bit BGRA rows as RGBA8.
- Positive-height bottom-up and negative-height top-down input are accepted.
- Rows use the standard 4-byte BMP row padding.
- Encode writes deterministic bottom-up Windows BMP with no color table.
- RGB-like input writes 24-bit BGR BMP. RGBA input writes 32-bit BGRA BMP.

## Non-Goals

- No indexed BMP, RLE-compressed BMP, bitfields BMP, OS/2 BMP headers, color
  tables, masks, high-depth BMP, metadata/profile preservation, color
  management, or full ImageMagick BMP semantics.
- No new command shapes beyond the existing `imx` identify/transcode/resize,
  resize-fit, and batch-convert surface.
- No `magick` alias, stdin/stdout, delegates, MagickCore, MagickWand,
  Homebrew/core, crates.io, Windows, or hosted macOS/iOS Actions.

## Required Evidence

- Codec unit tests for RGB24 padding, RGBA32 alpha, top-down input, and
  unsupported compression/bit-depth rejection.
- CLI tests for `.bmp`, `BMP:`, transcode, resize, resize-fit, batch-convert,
  prefix mismatch, missing path, same-path rejection, unsupported aliases, and
  lowercase prefix rejection.
- Golden, curated, malformed, fuzz-smoke, cargo-fuzz, and ImageMagick
  differential corpus coverage.
- Install, package, archive, benchmark, generated conformance, and Homebrew
  formula smoke evidence for BMP.
