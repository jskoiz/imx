# IMX v0.17.0 Self-Test and Diagnostics

`v0.17.0` adds one production-hardening slice: an offline installed-binary
self-test plus clearer CLI failure proof.

## Supported Surface

- `imx self-test`.
- Existing `imx --help`, `imx --version`, `identify`, two-path transcode,
  `resize`, `resize-fit`, and `batch-convert` behavior for BMP, FARBFELD,
  JPEG, QOI, PBM, PGM, PNG, and PPM.
- Release and package smoke that runs the self-test from the installed or
  unpacked binary, not only from a source checkout.

## Self-Test Contract

- Creates temporary fixtures for BMP, FARBFELD, JPEG, QOI, PBM, PGM, PNG, and
  PPM.
- Invokes the current installed binary through `std::env::current_exe()`.
- Verifies unprefixed and prefixed identify for every supported format.
- Verifies decode coverage by transcoding every fixture to PPM.
- Verifies encode coverage by transcoding a PPM fixture to every supported
  output format.
- Verifies same-format exact resize to `3x2`.
- Verifies same-format resize-fit inside `5x3`.
- Verifies batch-convert with resize-fit to each supported destination format.
- Prints progress lines ending in `self-test: passed` and exits `0` only when
  every step succeeds.
- Exits `1` with `error: self-test failed: ...` when any covered command fails.
- Deletes its temporary directory by default. `IMX_SELF_TEST_KEEP=1` preserves
  it for debugging.

## Diagnostics Contract

- Data, IO, malformed input, and validation failures exit `1` with
  `error: ...`.
- Usage and unsupported command shapes exit `2` and print usage.
- Tests cover unknown prefixes, mismatched prefixes, missing inputs,
  unsupported BMP compression, invalid resize geometry, same-path output,
  missing batch output directories, extra `self-test` arguments, and unsupported
  `convert`/`magick`/`mogrify` command shapes.
- Missing input and missing output directory errors name the missing path.
- Same-path output errors name both conflicting operands after prefix stripping.

## Non-Goals

- `imx self-test` is not an ImageMagick oracle, fuzz target, benchmark, or full
  conformance proof.
- No new formats, recursive batch mode, stdin/stdout, `magick` alias, full
  ImageMagick CLI parsing, delegates, profiles, color management, metadata
  preservation, MagickCore, MagickWand, Homebrew/core, crates.io, Windows, or
  hosted macOS/iOS Actions.

## Required Evidence

- Focused CLI tests for `imx self-test` and every diagnostic class above.
- `cargo fmt`, clippy, workspace tests, fuzz smoke, benchmark smoke, benchmark
  regression smoke, and ImageMagick differential corpus.
- Install, package, release-archive, generated conformance, and Homebrew formula
  smoke evidence that includes `imx self-test`.
- Published Linux release assets verified from GitHub `SHA256SUMS`.
