# IMX v0.15.0 Safe Batch Conversion

`v0.15.0` adds one explicit batch command:

```sh
imx batch-convert --to <FORMAT> --output-dir <dir> \
  [--resize <width>x<height>|--resize-fit <width>x<height>] \
  [FORMAT:]<input>...
```

The command is a safe orchestration layer over the existing single-file
surfaces. It uses the same decoders, encoders, exact uppercase input prefixes,
exact resize, and resize-fit behavior already proven for FARBFELD, JPEG, QOI,
PBM, PGM, PNG, and PPM.

## Contract

- `<FORMAT>` must be exactly `FARBFELD`, `JPEG`, `QOI`, `PBM`, `PGM`, `PNG`, or
  `PPM`.
- `--output-dir` must name an existing directory.
- Input arguments are literal shell-provided paths. IMX does not expand globs
  or walk directories.
- Output paths are `<output-dir>/<input-file-stem>.<target-extension>`, using
  `.ff`, `.jpg`, `.qoi`, `.pbm`, `.pgm`, `.png`, or `.ppm`.
- Optional `--resize` applies exact nearest-neighbor resize before encode.
- Optional `--resize-fit` applies aspect-preserving nearest-neighbor resize
  before encode.
- `--resize` and `--resize-fit` are mutually exclusive.
- Successful batch conversion is silent.

## Safety Rules

The full batch is preflighted before any output is written. IMX fails with
`error: ...` for missing inputs, unsupported target formats, invalid dimensions,
invalid or mismatched input prefixes, non-file inputs, duplicate derived output
paths, existing outputs, and output paths that resolve to the same file as an
input.

Batch outputs are no-overwrite. IMX does not rename collisions, append numeric
suffixes, or replace existing files. If decode, transform, or encode fails while
preparing the batch, previously prepared outputs are not committed.

## Non-Goals

This release does not add recursive directory walking, glob parsing beyond
shell-expanded arguments, overwrite mode, collision renaming, stdin/stdout,
watch mode, parallel execution, metadata preservation, color management, new
formats, full ImageMagick CLI parsing, a `magick` alias, MagickCore, MagickWand,
Homebrew/core, crates.io, Windows, or hosted macOS/iOS Actions.
