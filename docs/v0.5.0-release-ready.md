# IMX v0.5.0 Release-Ready Checklist

Status: release checklist. The `v0.5.0` tag must point at the approved release
commit, GitHub Release assets must be verified from the published
`SHA256SUMS`, and the Homebrew tap update must be verified before the release
goal is complete.

## Release Surface

- Binary: `imx`.
- Formats: FARBFELD, QOI, PBM, PGM, PPM.
- Commands: `imx identify <input>` and `imx <input> <output>`.
- Compatibility delta: deterministic same-format rewrites for the five
  supported formats when input and output paths differ.
- Planned hosted release targets: `x86_64-unknown-linux-gnu` and
  `aarch64-unknown-linux-gnu`.

## Approval Gate

Before publication, every release gate must pass. After publication, verify the
published archives, release notes, checksums, conformance assets, and tap update
from the release assets before claiming completion.

## Post-Publish Evidence

After approval and tag publication, verify the published Linux x86_64 and Linux
arm64 archives from GitHub Release assets, then update `jskoiz/homebrew-imx`
from the published `SHA256SUMS`.
