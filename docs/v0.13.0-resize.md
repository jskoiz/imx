# IMX v0.13.0 Bounded Resize Slice

Status: published Linux release and tap distribution. v0.13.0 release support is
claimed for the Linux archive targets present in the published GitHub
`SHA256SUMS` and the tap formula generated from those checksums, after
Linux-only release/archive smoke and tap smoke passed.

## Supported Surface

- Command: `imx resize <width>x<height> [FORMAT:]<input> [FORMAT:]<output>`.
- Formats: FARBFELD, JPEG, QOI, PBM, PGM, PNG, and PPM only.
- Prefixes: exact uppercase `FARBFELD:`, `JPEG:`, `QOI:`, `PBM:`, `PGM:`,
  `PNG:`, and `PPM:` keep the existing confirm-only behavior.
- Dimensions: lowercase `<width>x<height>` with non-zero unsigned 32-bit
  decimal values.

## Behavior Contract

Resize decodes the input image, applies center-sampled nearest-neighbor scaling
to exact dimensions, then runs the existing destination encoder. The source
coordinate for each destination axis is:

```text
floor(((2 * dst + 1) * src_size) / (2 * dst_size))
```

The coordinate is clamped to the last source row or column. IMX copies the full
decoded pixel value. It does not interpolate, composite alpha, preserve
metadata, perform color management, or choose a new bit depth. Existing encoder
rules still decide JPEG loss, QOI/Netpbm quantization, PBM thresholding, alpha
rejection, and metadata loss.

## Evidence Requirements

- Core image tests for format preservation, invalid dimensions, oversized
  output rejection, and center-sampled coordinate mapping.
- CLI tests for resize success across all seven formats, exact prefixes,
  invalid dimensions, unknown/missing/mismatched prefixes, extensionless output
  rejection, and same-path rejection.
- ImageMagick oracle proof against `-filter Point -resize <width>x<height>!`
  with exact decoded pixels for non-JPEG formats and existing JPEG tolerance
  metrics for JPEG output.
- Install, package, release-archive, conformance, and Homebrew formula smoke
  checks that exercise resize.

## Non-Goals

No new formats, stdin/stdout, batch mode, crop, rotate, percentages,
aspect-ratio shorthand, filters beyond nearest neighbor, metadata preservation,
color management, `magick` alias, full ImageMagick CLI parsing, delegates,
MagickCore, MagickWand, Homebrew/core, crates.io, Windows, or hosted macOS/iOS
GitHub Actions.
